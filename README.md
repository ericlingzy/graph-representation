# graph-representation
本研究的完整实验代码、原始数据和分析脚本均已作为补充材料提交。
具体包括：
实验主脚本 experiment_v2.py，包含完整的实验流程，所有随机种子均已固定；
实验输出数据（exp1_results.pkl、exp1_stats.json、exp2_base.pkl、
exp2_intervention.pkl、sweep_r.pkl、sweep_T.pkl）；
收敛曲线图和干预剂量-反应图。
运行环境为 Python 3.12，主要依赖 NumPy、SciPy、Scikit-learn、NetworkX、pythonlouvain
0.16、Statsmodels、Matplotlib。
实验在 Windows 工作站（64GB RAM）上完成，总运行时间约 6.5 小时。
